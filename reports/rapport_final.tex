\section{Introduction et Contexte}

\subsection{Présentation du projet}
Ce projet a été réalisé dans le cadre de l'atelier Machine Learning. L'objectif est de développer une solution complète d'analyse comportementale pour une entreprise e-commerce spécialisée dans la vente de cadeaux.

\subsection{Objectifs}
\begin{itemize}
    \item Analyser le comportement des clients pour mieux comprendre leurs habitudes d'achat
    \item Prédire le risque de départ des clients (churn)
    \item Développer un modèle de classification performant
    \item Créer une interface utilisateur pour la prédiction en temps réel
\end{itemize}

\section{Résultats et Recommandations}

\subsection{Principaux enseignements}
\begin{enumerate}
    \item La \textbf{date du dernier achat (Recency)} est le facteur le plus prédictif du churn (9.20\%)
    \item La \textbf{fréquence d'achat (Frequency)} est le deuxième facteur clé
    \item Le modèle atteint une précision de \textbf{91.9\%}, suffisante pour un déploiement en production
    \item La segmentation a identifié 4 groupes distincts avec des comportements très différents
\end{enumerate}

\subsection{Recommandations marketing}
\begin{itemize}
    \item \textbf{Cibler les clients à risque élevé} : Envoyer des offres personnalisées aux clients identifiés avec probabilité > 70\%
    \item \textbf{Réactiver les clients dormants} : Campagne d'emailing ciblée après 60 jours d'inactivité
    \item \textbf{Fidéliser les clients VIP} : Programme de fidélité avec avantages exclusifs
    \item \textbf{Suivi des clients occasionnels} : Newsletter engageante, recommandations produits
\end{itemize}

\section{Conclusion}

Ce projet a permis de :
\begin{itemize}
    \item Maîtriser l'ensemble de la chaîne de traitement en data science (exploration → préparation → modélisation → évaluation → déploiement)
    \item Appliquer des techniques avancées de prétraitement (imputation, encodage, normalisation)
    \item Développer un modèle de classification performant (91.9\% d'exactitude)
    \item Créer une segmentation client actionable avec K-Means (4 segments identifiés)
    \item Déployer une application web fonctionnelle avec Flask
\end{itemize}

Les résultats obtenus démontrent la faisabilité d'une solution de prédiction du churn client, permettant à l'entreprise d'anticiper les départs et d'optimiser ses stratégies marketing.
